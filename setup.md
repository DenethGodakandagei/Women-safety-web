# SheShield Project Setup

## Prerequisites
- MongoDB installed locally or a Cloud URI.
- Node.js (Latest LTS recommended).

## Backend Setup (5000)
1. Navigate to /backend.
2. Ensure MongoDB is running.
3. run `npm install` (already done by the setup).
4. run `npm start` to run in production mode or `npm run dev` for development.

## Frontend Setup (5173/3000)
1. Navigate to /frontend.
2. run `npm install` (already done by the setup).
3. run `npm run dev` to start the development server.

## Features implemented in this setup:
- Modern UI with Framer Motion and Glassmorphism.
- User and Emergency Contact data models.
- Secure backend foundation (CORS, Rate Limiting, NoSQL protection).
